<h1>Tipos básicos / Basic Types</h1>

<p>
    <h3>Type inference</h3>
    <strong>Tipagem DINÂMICA:</strong> Define o tipo da variável de acordo com o valor passado dentro da variável.
    Por exemplo: se for <code>let estado = "bahia";</code> a variável <strong>estado</strong> é definida automaticamente
    como do tipo String, e não pode ser alterada para outro tipo. <br>
    Oque acaba evitando erros comparado com o JavaScript, que deixaria adicionar um numero dentro de uma variável do tipo String.
</p>
